About

The Open Observatory of Network Interference (OONI) is a global observation network led by the Tor Project. They make observations targeting network tempering and publish the results available openly.

Journalists, technologists, and legal scholars can use these observations to help illustrate where instances of network tampering take place, indicating which parties may be responsible.

The goal of OONI is to detect the presence of network devices which interfere with network traffic and help expand understanding which types of content are being targeted for discrimination.

The architectural design of OONI allows third parties to contribute to the project by expanding tests centred around the detection of network traffic interference. After being subject to peer review, the tests can be executed against a set of experiment and control networks. These tests permit the detection of network anomalies using an experiment-control methodology.

A mismatch between the experiment and control data is not always indicative of network manipulation, in many protocols this is a clear indication that some form of tampering has taken place.

ooni-probe attempts to detect the presence of network traffic manipulation from the perspective of a given edge network probe using a blend of traffic manipulation and content blocking tests.

The goal of the traffic manipulation tests are to detect the presence of a network device which performs traffic manipulation to modify how a given user sees a particular network resource. Tests in this category may look for evidence of DNS spoofing, HTTP header manipulation, captive portals, and route tampering.

Content blocking tests aim to access content which may be blocked by a given actor. Examples of tests in this category include DNS consistency testing, Tor bridge reachability testing, and tests which attempt to connect to one, or more censored resources.
